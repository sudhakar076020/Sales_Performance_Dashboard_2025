# 📊 Sales & Profit Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Sales & Profit Analysis Dashboard** developed using **Power BI**.  
The dashboard provides insights into business performance across regions, categories, sub-categories, and time.

The objective of this project is to analyze sales trends, profitability, and regional performance to support data-driven decision-making.

---

## 🗂 Dataset Information

The dataset contains **100 rows** with the following columns:

- Order ID  
- Order Date  
- Ship Date  
- Delivery Date  
- Customer Name  
- Region  
- State  
- Category  
- Sub-Category  
- Product Name  
- Sales  
- Quantity  
- Profit  

The dataset was prepared in Excel and imported into Power BI for visualization and analysis.

---

## 📈 Dashboard Components

### 🔹 KPI Cards
The dashboard includes four Key Performance Indicators (KPIs):

- **Total Sales** – Overall revenue generated  
- **Total Profit** – Net profit earned  
- **Total Quantity** – Total units sold  
- **Total Orders** – Total number of orders processed  

These KPIs provide a quick summary of overall business performance.

---

### 🔹 1️⃣ Sales & Profit by Region (Clustered Column Chart)

- Compares regional performance  
- Identifies the highest revenue-generating region  
- Helps analyze geographic contribution  

---

### 🔹 2️⃣ Monthly Sales Trend (Line Chart)

- Displays sales performance over time  
- Shows growth and fluctuation patterns  
- Helps identify seasonal trends  

---

### 🔹 3️⃣ Sales by Category (Donut Chart)

- Shows percentage contribution of each category  
- Helps understand which product segment drives revenue  

---

### 🔹 4️⃣ Profit by Sub-Category (Stacked Bar Chart)

- Displays profitability across sub-categories  
- Helps identify high-margin product groups  

---

## 🎛 Interactive Features

The dashboard includes slicers for:

- **Region**
- **Category**
- **State**

These slicers allow dynamic filtering of all visuals, making the dashboard interactive and user-friendly.

---

## 📊 Key Insights

- The North region generates the highest sales.
- Technology category contributes the largest share of revenue.
- Profit varies significantly across sub-categories.
- Monthly sales show fluctuations, indicating possible seasonal impact.
- The business maintains a healthy overall profit margin.
